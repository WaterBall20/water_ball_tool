# 水球包文件二进制索引文件设计

_创建： 2026/02/27 14:18_

# 核心结构

核心结构分为**属性**、**结构**和**虚拟文件元数据**

# 属性 Attribute

表示索引数据的属性,属性是全部常驻于内存中

## 格式

| 类型                                                 | 字节数 | 说明                               |
| ---------------------------------------------------- | ------ | ---------------------------------- |
| 格式版本 version                                     | 2      | 表示当前索引版本                   |
| 格式兼容版本 ..compatible                            | 2      | 表示兼容最低的解析器版本           |
| 布尔数据                                             | 1      | 按二进制表示的布尔值               |
| 空数据列表的文件指针位置 empty_data_pos              | 8      | 表示空数据段存放的文件指针位置     |
| 清单空数据列表的文件指针位置 manifest_empty_data_pos | 8      | 表示清单空数据段存放的文件指针位置 |
| 所有文件数 file_count                                | 8      | 表示有多少个文件，不包括目录       |
| 所有目录数 dir_count                                 | 8      | 表示有多少个目录                   |
| 数据大小 data_len                                    | 8      | 表示实际数据占用                   |

### 布尔数据

| 类型             | 二进制位序 | 说明             |
| ---------------- | ---------- | ---------------- |
| 启用写实复制 cow | 0          | 是否启用写实复制 |

### 空数据列表

表示所有内部空隙的列表

| 类型                        | 字节数     | 说明                                           |
| --------------------------- | ---------- | ---------------------------------------------- |
| 空数据列表数量              | 8          | 表示空数据有多少个                             |
| 空数据列表 empty_data_pos_s | 取决于占用 | 表示空数据列表，存放多个空数据的起始位置和长度 |

#### 空数据项

| 类型 | 字节数 | 说明                 |
| ---- | ------ | -------------------- |
| 位置 | 8      | 表示空数据的起始位置 |
| 长度 | 8      | 表示空数据的长度     |

# 结构 PackStruct

结构用于**树形**连接，即类似于文件夹和文件的结构，单个结构数据是连续的，但各个结构数据不是连续的，而是通过文件指针连接。

## 格式

由连续的**结构项**组成

### 结构项 PackStructItem

结构项是分类型的

| 类型                                      | 字节数         | 说明                                                                                 |
| ----------------------------------------- | -------------- | ------------------------------------------------------------------------------------ |
| 长度                                      | 8              | 64位表示当前结构项的长度                                                             |
| 类型                                      | 1              | 表示此结构项的类型                                                                   |
| 名称长度                                  | 2              | 表示名称占用多少字节                                                                 |
| 名称 name                                 | 取决于占用大小 | 表示名称                                                                             |
| 虚拟文件元数据的文件指针位置 metadata_pos | 8              | 表示虚拟文件的元数据的文件指针位置,<br/>注意：符号链接**没有**元数据，此字段将不存在 |
| 类型特有数据                              | 取决于类型     | 表示类型特有的数据                                                                   |

#### 类型及特有数据 PackStructItemType

##### 文件(0x0) File

文件的类型值为0x0,文件不包含结构类型特有数据。

##### 目录(0x1) Dir(PackStructItemDir)

目录的类型值为0x1

**特有数据**

| 类型                               | 字节数 | 说明                         |
| ---------------------------------- | ------ | ---------------------------- |
| 结构的文件指针位置 struct_file_pos | 8      | 表示该目录结构的文件指针位置 |

##### 符号链接(0x2) Link

符号链接的类型值为0x2

**注意：** 符号链接目前还没有定义

# 虚拟文件元数据 PackFileMetadata

虚拟文件的元数据信息，所有元数据信息不是连续的。

元数据是分类型的，目前分**文件**、**目录**和**符号链接**。

## 格式

| 类型              | 字节数     | 说明                                     |
| ----------------- | ---------- | ---------------------------------------- |
| 类型              | 1          | 表示文件类型                             |
| 布尔数据          | 1          | 按二进制表示的布尔值                     |
| 长度 len          | 8          | 表示虚拟文件占用的长度                   |
| 修改时间 modified | 16         | 128位表示修改时间                        |
| 类型特有数据      | 取决于类型 | 存放类型特有数据，类型在布尔数据内部表示 |

## 布尔数据

| 类型         | 二进制位序 | 说明             |
| ------------ | ---------- | ---------------- |
| 启用写实复制 | 0          | 是否启用写实复制 |

## 类型特有数据 PackMetadataType

### 文件(0x0) File(PackMetadataFile)

| 类型               | 字节数     | 说明                           |
| ------------------ | ---------- | ------------------------------ |
| 哈希值算法类型     | 1          | 表示哈希算法类型，目前未定义   |
| 哈希值长度         | 1          | 表示哈希值占用长度             |
| 哈希值             | 取决于算法 | 表示哈希值                     |
| 已分配数据列表数量 | 8          | 表示该文件已分配数据列表的数量 |
| 已分配数据列表     | 取决于数量 | 表示该文件已分配数据列表       |

#### 已分配数据项

和空数据位置项相同

| 类型 | 字节数 | 说明                 |
| ---- | ------ | -------------------- |
| 位置 | 8      | 表示空数据的起始位置 |
| 长度 | 8      | 表示空数据的长度     |

### 目录(0x1) Dir(PackMetadataDir)

| 类型     | 字节数 | 说明                               |
| -------- | ------ | ---------------------------------- |
| 文件数量 | 8      | 表示当前目录有多少文件，不包含目录 |
| 目录数量 | 8      | 表示当前目录有多少目录             |
